## Задание: Опиши котика

Твоя задача — создать класс `Cat`, который будет описывать котика. У каждого котика есть имя и возраст. Котик умеет мяукать и рассказывать о себе.

### Что нужно сделать:

1. Создай класс `Cat`.
  
2. Добавь конструктор `__init__`, который принимает имя и возраст котика и сохраняет их.
  
3. Добавь метод `meow`, который печатает: `"Мяу! Меня зовут [имя]"`
  
4. Добавь метод `info`, который печатает: `"Котик [имя], возраст [возраст] лет"`
  

### Пример работы программы:

```python

# Создаем котика

my_cat = Cat("Барсик", 3)

# Котик мяукает

my_cat.meow()

# Котик рассказывает о себе

my_cat.info()
```
**Что должно вывестись:**
```
# Мяу! Меня зовут Барсик
# Котик Барсик, возраст 3 лет
```
