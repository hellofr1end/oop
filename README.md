## Задание: Опиши котика

Твоя задача — создать класс `Cat`, который будет описывать котика. У каждого котика есть имя и возраст. Котик умеет мяукать и рассказывать о себе.

### Что нужно сделать:

1. Создай класс `Cat`.
  
2. Добавь конструктор `__init__`, который принимает имя и возраст котика и сохраняет их.
  
3. Добавь метод `meow`, который печатает: `"Мяу! Меня зовут [имя]"`
  
4. Добавь метод `info`, который печатает: `"Котик [имя], возраст [возраст] лет"`
  

### Пример работы программы:

```python

# Создаем котика

my_cat = Cat("Барсик", 3)

# Котик мяукает

my_cat.meow()

# Котик рассказывает о себе

my_cat.info()
```
**Что должно вывестись:**
```
# Мяу! Меня зовут Барсик
# Котик Барсик, возраст 3 лет
```

## Задание: Библиотека книг

Создай класс `Book`, который описывает книгу. У книги есть название, автор и статус — взята ли книга в библиотеке или нет.

Книга умеет:

- Показывать информацию о себе
  
- Менять статус (выдана/возвращена)
  

---

### Что нужно сделать:

1. Создай класс `Book`.
  
2. В методе `__init__` сохрани название, автора и добавь атрибут `is_borrowed` со значением `False` (по умолчанию книга свободна).
  
3. Создай метод `info`, который печатает: `"Название: [название], Автор: [автор], Статус: [свободна/выдана]"`
  
4. Создай метод `borrow`, который:
  
  - Если книга свободна — меняет статус на `True` и печатает: `"Вы взяли книгу [название]"`
    
  - Если книга уже выдана — печатает: `"Книга [название] уже выдана"`
    
5. Создай метод `return_book`, который:
  
  - Если книга выдана — меняет статус на `False` и печатает: `"Вы вернули книгу [название]"`
    
  - Если книга и так свободна — печатает: `"Книга [название] и так в библиотеке"`
    

```python
book1 = Book("Мастер и Маргарита", "Булгаков")
book2 = Book("Война и мир", "Толстой")

book1.info()
book2.info()

book1.borrow()
book1.borrow() # пробуем взять ещё раз
book1.info()

book1.return_book()
book1.return_book() # пробуем вернуть ещё раз
book1.info()

```
