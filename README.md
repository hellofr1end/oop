# Задание 3: Кофейня

Создай два класса: `Coffee` и `Customer`. Кофейня работает так: у каждого кофе есть название и цена. Посетитель может заказать кофе, если у него хватает денег.

---

### Что нужно сделать:

**Класс `Coffee`:**

- `__init__(self, name, price)` — сохраняет название и цену кофе
  
- `info(self)` — печатает `"Кофе [название], цена [цена] руб."`
  

**Класс `Customer`:**

- `__init__(self, name, money)` — сохраняет имя посетителя и сколько у него денег
  
- `info(self)` — печатает `"Посетитель [имя], баланс: [деньги] руб."`
  
- `order(self, coffee)` — принимает объект кофе и пытается его купить:
  
  - Если денег хватает — вычитает цену кофе из баланса и печатает `"[имя] купил(а) [название кофе]. Остаток: [деньги] руб."`
    
  - Если денег не хватает — печатает `"У [имя] не хватает денег на [название кофе]"`
    

---

### Пример работы программы:

```python
Создаем кофе

espresso = Coffee("Эспрессо", 150)
latte = Coffee("Латте", 220)

# Создаем посетителей

# Показываем информацию

print("---")

# Делаем заказы

# Проверяем балансы
```
